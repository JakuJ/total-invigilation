This page knows you're looking away

This is a collection of scripts that detect user inactivity and leaving the page. In particular, it knows if you:

  • Minimize the window
  • Switch to another tab
  • Switch to another program
  • Do not show any mouse or keyboard activity for over 20 seconds
Every time you do any of these, you will receive a notification. They are pretty annoing so you can disable them if you want:
